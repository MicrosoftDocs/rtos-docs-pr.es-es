# <a name="appendix-a---azure-rtos-netx-secure-returnerror-codes"></a>Apéndice A: Códigos de error y devolución de Azure RTOS NetX Secure

## <a name="netx-secure-tls-return-codes"></a>Códigos de devolución de TLS de NetX Secure

En la tabla 1 que figura a continuación se enumeran los posibles códigos de error que pueden devolver los servicios de TLS de Azure RTOS NetX Secure. Tenga en cuenta que los servicios también pueden devolver códigos de error de TCP/IP: los valores de TLS comienzan en 0x101, mientras que los de TCP/IP son inferiores a 0x100. Los valores devueltos de X.509 comienzan en 0x181. Consulte la documentación de TCP/IP de NetX para obtener información sobre los valores devueltos de TCP/IP y consulte a continuación los valores de X.509.
